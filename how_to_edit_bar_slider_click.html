Flash XML Bar Slider



Bar Slider Click

Adjust width and height values of the Flash file ( flashmo_255_bar_slider_click.swf ) in the SWF embedded HTML file. Always add 30 pixels to the height of SWF.

Example:

  • set SWF height value to 330 if the image height is 300 pixels
  • set it to 430 if the image height is 400 pixels because height of the navigation bar is 30 pixels

Default XML file name

xml_file = flashmo_255_photo_list.xml ( or any file name can be set in SWF embedded HTML tags )

Example: flashmo_255_bar_slider_click.swf?xml_file=flashmo_255_photo_list.xml

Variables in XML <config> tag

hover_effect = true by default ( or false ) for mouse pointer over image area

auto_play = true by default ( or set it as false )

auto_play_duration = 3 seconds by default ( or any number of seconds )

tween_duration = 0.72 seconds by default ( or any number of seconds )

tween_delay = 0.16 seconds by default ( or any number of seconds )

Transition Effects in XML <transition> tag

sequence = first | last | inner | outer

direction = left | right | up | down

slice = 3 | 4 | 5 | 6 | 7 | or any positive integer number ( odd numbers are preferred if sequence is set to inner or outer )

effect = easeInOutQuart | easeInOutCubic | easeInOutExpo | easeInOutSine ( or see below Tweener transition cheat sheet )
